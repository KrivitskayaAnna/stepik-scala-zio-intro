# stepik-scala-zio-intro
Проект к курсу по Scala ZIO на Stepik https://stepik.org/course/292230

На 26 июня 2026 курс еще не опубликован

Структура репозитория:

- /lessons/lessonN содержат код к видеоурокам

- /tasks/lessonN содержат задачи к урокам 

Некоторый код в задачах закомментирован, тк он не компилируется без решения задачи, при решении его необходимо раскомментировать

- /solutions/lessonN содержат мои решения задач к урокам
